Max Ellis is a Senior Software Engineer building resilient systems with a firefighter's mindset. He specializes in distributed data systems, real-time streaming pipelines, AI/RAG systems, cloud infrastructure (AWS), and system reliability and performance.

High-velocity Senior Software Engineer with a proven track record of architecting scalable data platforms and automating mission-critical infrastructure.
